Hace tiempo empecé a aprender Angular, venía de Ember y antes de probar AngularJS. Para ser sincero Angular es algo difícil de aprender y necesitas aplicar tiempo, pero una ventaja que ofrece es lo fácil que se vuelve manejar un proyecto que se vuelve grande.

## Pre-requisitos

### 1. NodeJS

Lo podemos descargar desde su [página oficial](https://nodejs.org/es/download/)

**Instalación simple** con siguiente, siguiente, ...

### 2. angular/cli

Para los siguientes pasos usaremos la terminal, puedes usar la que quieras

    npm install -g @angular/cli

## Creando la aplicación de Angular

Ahora para esta parte abriremos la consola donde queramos meter nuestro proyecto, pues nos generará todos los archivos dentro, en mi caso he creado una carpeta llamada Angular.

    ng new aplicacion-test

Nos hará escoger algunas opciones, si no sabes muy bien de que se habla dale palante a todo, luego se puede cambiar.

## Arrancando la aplicación

Para arrancar la aplicación de Angular tenemos que entrar en la carpeta que se ha creado, en este caso será **aplicacion-test**

    npm install
    npm start

Si ahora abrimos un navegador y nos dirigimos a [http://localhost:4200](http://localhost:4200) veremos nuestra aplicación funcionando.
